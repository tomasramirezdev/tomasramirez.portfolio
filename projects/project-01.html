Project Overview

Led the entire UX/UI design process for a premium race car rental platform focused on improving conversion rates and user experience flow. The project aimed to create an intuitive booking system for high-end racing experiences.

Design Process

01
User Research
Conducted extensive user interviews with racing enthusiasts and luxury car renters to understand booking behaviors and pain points.
02
Competitive Analysis
Analyzed existing car rental platforms and racing experiences to identify opportunities for differentiation.
03
Wireframing
Created low-fidelity wireframes focusing on the booking flow and vehicle selection process.
04
Prototyping
Developed interactive prototypes in Figma to test user flows and gather feedback.
05
UI Design
Designed high-fidelity interfaces with a premium aesthetic that matches the luxury racing experience.
06
User Testing
Conducted usability testing sessions to validate design decisions and optimize the user experience.

Key Challenges & Solutions

Challenge: Complex booking flow with multiple variables (car selection, track availability, insurance options, etc.)

Solution: Implemented a step-by-step wizard interface with clear progress indicators and the ability to save preferences.

Challenge: High-value transactions requiring trust and confidence

Solution: Added detailed vehicle specifications, high-quality imagery, and transparent pricing with no hidden fees.

Challenge: Mobile-first experience for users booking on-the-go

Solution: Responsive design with touch-friendly interactions and simplified mobile booking flow.
